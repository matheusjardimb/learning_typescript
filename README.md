# Learning typescript

Experiments from the Udemy's course [Understanding Typescript](https://www.udemy.com/course/understanding-typescript/)

## Setting up env

```bash
nvm use
npm -g install typescript
npm i
```

## Running the project

```bash
# Watch mode
# tsc app.ts -w

# Uses tsconfig.json to load all .ts files
tsc -w

npm run start

```
